Rotating Fixture Setup

Problem

A machine vision setup which checks for errors in a workpiece. For the output to be accurate the position of the workpiece is to be same with negligible misposition. But when the workpieces are placed manually inside the system, they are more prone to errors.


User

Assembly line workers in the quality control section.


My Role

To design a fixture which can be fixed to a rotating disk upon which the workpiece can be placed. The fixture is in such a way to match the workpiece perfectly so that there will be no disposition and the system can be accurate.


Design Process

The metrology part was the real challenge with this project because designing a fixture for a confidential piece without dimensions is a tedious job. The workpiece was measured thoroughly using measuring devices and the sketch of the workpiece was first obtained. Then the fixture was designed to hold the workpiece in position for the inspection. Also, easy replacement of the workpiece was checked to be perfect to ease the work of the user.
